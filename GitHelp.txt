команды для регистрации данных
1)git config --global user.name "имя в гитхабе"
2)git config --global user.email "your email"

команды для одного проекта

3)git init - (активирует гит)отслеживает все изменения
4)git remote add origin (link_to_repository, ссылка на репозиторий) | добавление репозитория

5)git status | статус репозитория

6)git add .  | добавление файлов в буфер обмена
7)git commit -m "massage" | сохранение всего из буфера обмена

8)git push origin main | отправка сохранений на GitHub

9)git branch | посмотреть все ветки
10) git branch new_branch | создать новую ветку
11) git checkout new_branch | переключиться на новую ветку
12) git switch new_branch | переключиться на новую ветку
